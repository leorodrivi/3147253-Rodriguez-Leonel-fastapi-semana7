# Guía de Performance - Centro de Yoga Paz Interior

## Objetivo

Esta guía proporciona mejores prácticas y estándares para mantener y mejorar el performance del sistema de gestión de clases de yoga.

## Métricas de Performance Objetivo

### SLAs (Service Level Agreements)
- **Tiempo de respuesta promedio:** < 200ms
- **Tiempo de respuesta p95:** < 500ms
- **Disponibilidad:** 99.9%
- **Tasa de error:** < 1%

### Límites de Resource Usage
- **Memoria RAM:** < 512MB
- **CPU:** < 70% utilization
- **Storage cache:** < 100MB
- **Concurrent connections:** < 1000
